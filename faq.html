Does this application offer non-alcoholic drinks?

Yes! drinKit offers drinks with both alcoholic and non-alcoholic content so hopefully there's something for everyone! You can use our filtering system to search only for non-alcoholic drinks if you don't want drinks containing alcohol to appear in your search.

I have a child who’d like to try making a drinKit recipe. Can I stop them accessing alcoholic content at all?

Yes! In the Settings menu, toggle ‘Kid Mode’ to automatically filter out any recipes that contain alcohol.

What devices is drinKit supported on?

drinKit can run on all iOS devices version 8+ and Android devices version 4.4+. There’s no difference between features or content either!

Is drinKit free?

Yes! drinKit can be downloaded for no cost from the Apple or Google Play store, and there’s no sneaky hidden costs needed to unlock content either.

I'm having difficulty reading the application content. Can I change the text size?

Yes! On the Settings menu, you can increase or decrease the font size of all the text on drinKit's pages.

Can drinKit be used offline?

To keep you updated with the latest details of our great drink recipes, drinKit requires a constant Internet connection to view our full selection. However, any drinks you’ve favourited can be kept offline for easy access.

How is drinKit built?

drinKit is built using the Ionic Framework, an open source SDK licensed under MIT that lets us built applications for Android and iOS. The application is written with AngularJS and styled with HTML and CSS.

How does drinKit store and load recipes?

All of drinKit’s recipes are stored in an external database; no hard coding here. drinKit loads content to display based on your requests, made and answered through the use of MySQL.
